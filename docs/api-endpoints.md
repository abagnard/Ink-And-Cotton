#JSON API
---
###Users

###Session
  GET /api/session
  POST /api/session

###Products
  GET /api/products

  DELETE /api/products/:id

  GET /api/reviews
  POST /api/reviews
  PATCH /api/reviews/:id
  DELETE /api/reviews/:id

###CartItem

  POST api/cart_items
  PATCH api/cart_items/:id
